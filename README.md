## Installation-


## ENV
Create a .env file in the main dir  and add -

```bash
NEXT_URL=http://localhost:3000/ [Next local Address]

TMDB_API_KEY=[Your TMDB key]

TMDB_READ_ACCESS_KEY=[Your Access Key]
```




## To run 
```bash
pnpm  i

pnpm dev
```
